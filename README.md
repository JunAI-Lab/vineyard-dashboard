# RJN Cua Vineyards · Wage Dashboard (Demo)

Interactive, bilingual (中文 / English) wage dashboard. **Worker names are masked** for privacy.

**Live:** https://junai-lab.github.io/vineyard-dashboard/

Single-page app: filter by category / patch / date, donut + bar charts, sortable payroll
table, per-worker drill-down, dark/light + 中/EN toggles, mobile-friendly card view. System fonts, no external dependencies.
